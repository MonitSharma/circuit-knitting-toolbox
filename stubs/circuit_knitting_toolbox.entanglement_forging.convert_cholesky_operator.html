circuit_knitting_toolbox.entanglement_forging.convert_cholesky_operator¶

convert_cholesky_operator(operator, ansatz)[source]¶

Convert the Cholesky operator (ListOp) into the entanglement forging format.

Parameters:

  • operator (ListOp) – A ListOp containing the single-body Hamiltonian followed by the Cholesky operators shape: [single-body hamiltonian, cholesky_0, …, cholesky_N]

  • ansatz (EntanglementForgingAnsatz) – The ansatz for which to compute expectation values of operator. The EntanglementForgingAnsatz also contains the bitstrings for each subsystem

Return type:

EntanglementForgingOperator

Returns:

An EntanglementForgingOperator object describing the decomposed operator
