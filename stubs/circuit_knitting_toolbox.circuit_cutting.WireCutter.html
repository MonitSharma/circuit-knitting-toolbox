circuit_knitting_toolbox.circuit_cutting.WireCutter¶

class WireCutter(circuit, service=None, options=None, backend_names=None)[source]¶

Class to hold the main cutting functions.

This is the main class of user interaction for Wire Cutting. This class manages all the key user functions such as decomposing (cutting) the circuit, evaluating the subcircuits, recomposing results to reconstruct the full output, and verifying that output against the ground truth distribution.

- circuit

original quantum circuit to be cut into subcircuits

Type:

QuantumCircuit

- service

the runtime service used to simulate or execute the subcircuits on real systems

Type:

  • service (-) – the runtime service used for executing the subcircuits

  • options (-) – options for executing in the session

  • backend_names (-) – the backends to execute the subcircuits on
