circuit_knitting_toolbox.entanglement_forging.cholesky_decomposition¶

cholesky_decomposition(problem, mo_coeff=None, orbitals_to_reduce=None)[source]¶

Construct the decomposed Hamiltonian from an input ElectronicStructureProblem.

Parameters:

  • problem (-) – An ElectronicStructureProblem from which the decomposed Hamiltonian will be calculated

  • mo_coeff (-) – The coefficients for mapping to the MO basis. If None, the input integrals will be assumed to be in the MO basis

  • orbitals_to_reduce (-) – A list of orbital indices to remove from the problem before decomposition

Return type:

tuple[ListOp, float]

Returns:

Tuple containing the cholesky operator and the energy shift resulting from decomposition

Raises:

- ValueError – The input ElectronicStructureProblem contains no particle number information.
