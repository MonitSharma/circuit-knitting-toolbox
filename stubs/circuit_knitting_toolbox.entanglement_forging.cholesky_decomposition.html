circuit_knitting_toolbox.entanglement_forging.cholesky_decomposition¶

cholesky_decomposition(problem, orbitals_to_reduce=None)[source]¶

Construct the decomposed Hamiltonian from an input ElectronicStructureProblem.

Parameters:

  • problem (-) – An ElectronicStructureProblem from which the decomposed Hamiltonian will be calculated.

  • orbitals_to_reduce (-) – A list of orbital indices to remove from the problem before decomposition.

Return type:

Tuple[ListOp, float]

Returns:

  • Tuple containing

    • cholesky_operator (ListOp): A list of operators representing the decomposed Hamiltonian. shape: [single-body hamiltonian, cholesky_0, …, cholesky_N]

    • freeze_shift (float): An energy shift resulting from the decomposition. This shift should be re-applied after calculating properties of the decomposed operator (i.e. ground state energy).
