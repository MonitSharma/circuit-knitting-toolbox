class EntanglementForgingKnitter(ansatz, service=None, backend_names=None, options=None)[source]¶

A class which performs entanglement forging and returns the ground state energy and Schmidt coefficients found for given ansatz parameters and Schmidt coefficients.

  • service (-) – The service used to spawn Qiskit primitives and runtime jobs
